# 平台撒钱时，你不一定要去捡

最近 Virtuals Protocol 出了一个 AI Agent 交易赛：你质押 VIRTUAL 代币，猜哪个 Agent 的交易表现最好。所有人的质押汇成一个公共奖池。轮次结束后，奖池的 95% 按「你的得分 × 你的质押量」占全场总和的比例重新分配。平台无论如何先抽走 5%。

X 上有个大学生，5 天赚了 $1500。

我也是 21 岁，我能闻出来他身上的那股稚嫩味。

我的第一反应不是「怎么赚的」，而是「为什么我看了之后觉得很累」？这不就有点像主人给自家狗狗们撒粮食，让它们去争抢吗？这他妈不是清高——是算账。

## 这钱到底从哪来

拆开看，利润来源只有三个：

1. **平台抽水后的奖池再分配** — 你赚的来自比你更差的参与者的亏损
2. **信息差** — 比别人更早发现谁会入选、谁的数据在变好
3. **注意力溢价** — 新叙事吸引的新钱抬高了 Agent 代币价格

公式更直白：

```
你的利润 = 总奖池 × 95% × (你的得分 × 你的质押) / Σ(所有人得分 × 质押) - 你的质押
```

**没有任何一层在创造新价值。** 这是一个封闭的资金再分配系统。平台先抽 5%，剩下的在参与者之间转移。

同样适用于空投。Binance Research 和 Dragonfly 的 2025 年行业报告都指向同一个结论：空投已经从奖励早期真实用户，演化为 sybil farming、女巫检测、分配不透明、内幕分配的复杂博弈。farming 越普及，空投促进真实使用的效果就越被稀释。你学到的「刷交互」技巧高度依赖具体项目、具体规则、具体窗口期。项目一变，经验立刻贬值。

## 加密行业真正赚钱的地方不靠这个

按 2025 年协议收入口径，头部收入长这样：

- **稳定币利差/储备收益** — Tether $52 亿协议收入。四大稳定币发行商合计 $83 亿，占前十大协议的 65.7%。
- **交易手续费** — Coinbase 全年交易收入 $41 亿，订阅与服务收入占比已达 41%。
- **底层链手续费** — Tron $35 亿，Solana、Ethereum 紧随其后。

而头部**声量**长这样：投机、喊单、空投、meme、项目营销、币价叙事。

两者的重叠很小。

最赚钱的不需要你关注。最需要你关注的往往不赚钱——它要的是你的注意力和流动性。

## 「主人撒粮」不是阴谋论，是商业模式

平台的激励设计本质上是用代币/奖池购买用户的：注意力、流动性、参与度、社交证明、X 和社交媒体的声量。

用户以为自己是在「赚奖励」。平台其实是在买增长。

CoinMarketCap 的分析也指出，Virtuals「最高每月 $1M 奖励」的叙事能快速拉来注意力，并把它和 VIRTUAL 代币的短期价格波动绑在一起。

这不是阴谋论。这是标准的平台增长策略——OKX 和 Binance 早年也做过。但认识这一点很重要：**你不是在参与一个公平的游戏，你是在一个被精算过的 incentive system 里当增长单位。**

## 真正让你累的不是体力

这类游戏吃的不是时间，是**注意力主权**。

它会不断要求你：盯着谁发了新规则、盯着谁进场了、盯着奖池变了没、盯着实时 PnL、盯着有没有漏洞可以薅、盯着自己会不会变韭菜。

循环长这样：找机会 → 判断规则 → 怕错过 → 小额下注 → 等结果 → 复盘盈亏 → 下一轮继续。

不难，但碎。不重，但烦。不一定亏钱，但会亏掉主线注意力。

你的大脑已经识别到了：这东西不像在建设你的主线资产，而是在偷你的带宽。

## 你到底在干嘛

把 Virtuals Degen 和自己搭一个 trading agent 放在一起看：

Virtuals Degen 这边：你拥有的是赌对了的短期利润。不可复利，得靠重新找下一个局。失败后剩下亏损——和被设计的感觉。你几乎只能控制进出时机。技能不可迁移。

自己搭 Agent 那边：你拥有一个系统、策略、判断框架。可复利，系统迭代积累。失败后剩下代码、数据、经验、技能。你控制策略、风险、架构、边界。技能可迁移到任何市场。

最大区别：**Agent 即使不赚钱，你还是在生产。Degen 赚了钱，你什么都没生产。**

这个框架不只适用于加密。

## 「但这钱确实好赚啊」

这是最强的反对意见。那个大学生 5 天就赚了 $1500。

对，短期确实可能赚钱。信息差够大、进得够早、出得够快的人，能从平台获客预算和其他参与者手里提取利润。

但这句话混淆了「能赚钱」和「值得把时间投进去」。

5 天赚 $1500 意味着什么？如果是可重复、可积累的，当然值。但它不可重复——下一轮的规则、窗口、竞争格局都不一样。上一个 5 天的经验，不能让你在下一个 5 天赚更多。

花同样的 5 天：搭一个 agent 的风险引擎、写一篇可发表的市场分析、优化一个数据管道——这些东西 5 年后仍然在为你工作。

**「好赚」是短视的衡量标准。** 真正的问题是：5 年后回头看，这 5 天产生了什么？

## 什么情况下可以碰

默认不参与。

除非这个活动能变成你主线生产的一部分——比如你能拿它的数据训练模型、借它的机制写一篇分析、或者你本来就该练的风险控制能力恰好能用它当沙盘。而且钱包必须隔离，小本金，不能碰任何需要签名的陌生合约。

仅 2025 年，EVM 链上的签名钓鱼就造成 $83.85M 损失，波及 10.6 万受害者——还只是可追踪的 drainer 数据。每多一个你签名的陌生合约，就多一层风险。

**否则，不碰。**

## 三个问题

面对任何「快钱」机会时，问自己：

1. **赚钱之后呢？** — 我手里多了什么？（不只是钱）
2. **不赚钱之后呢？** — 我手里剩下什么？
3. **5 年后回头看，这段时间投得值吗？**

三个答案都不够好，不碰。

硅谷王川有一段话，我一直很认同：

> 挖空心思去研究、捡区块链上的空投，基本上属于回报递减、无法积累的行为。而且可能遇上恶意攻击，错误授权，把自己的 token 都弄丢了。
>
> 能够产生复利的事情往往有个特点：你今天创作的产出，可以成为第二天创作的输入。这样基础越来越厚，创作成果就越滚越大，最后会涌现出一个你自己都不可能预先设计出来的、有意思的东西。

## 这不是观点，是决策架构

我不是清高。我也不是在做内容。

我是给自己搭了一个判断框架——下次平台撒粮的时候，翻开就知道自己早就想清楚了。

这类钱会把你从「建设资产的人」，拉回「追逐奖励的人」。它可能赚钱，但不一定让你变强。它可能是机会，但也可能是伪装成机会的注意力税。

所以我的默认策略：不追空投、不追任务、不追短期奖池、不追平台撒粮——除非它能服务于我的主线生产：产品、代码、内容、认知、系统、资产。

**这钱可以赚，但不值得成为你的主线。你真正该赚的，是能让明天的你更强的钱。**

场外靠技能生血，场内靠复利养血。
